# alv_routing demo

Loosely made on the Air-Launch-Vehicle (ALV) made to loosely represent a 
MALD-like capability (See references to unclassified sources in the two txt files).
* alv_demo - This demo illustrates how to give a route to a ALV upon firing using the guidance computer.
* alv_demo_with_jammer - Adds a jammer to the ALV.
