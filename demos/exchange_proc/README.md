# exchange_proc demo

This is an example scenario to demonstrate a tanker-receiver interchange.  
*input_for_combined_tanker_to_receiver.txt* is the input file. The exchange was successful if:
1. The tanker instance started with 9000 kg of Payload, and off-loaded 5000 kg, to finish with 4000 kg Payload.
1. The receiver instance started with 3000 kg of Fuel, and on-loaded 5000 kg, to finish with 8000 kg Fuel.
