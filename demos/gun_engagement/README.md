# gun_engagement demo

A script that creates a bullet weapon, launch computer, and then runs a test scenario on the bullet.
The bullet demo runs at 100 Hz. It is not intended for mission level analysis.
Future expansion could include bullet dispersion and bullet muzzle velocity variance.
